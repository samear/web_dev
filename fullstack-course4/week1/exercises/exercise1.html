Font Style and Images

When we don't want to use CSS for styling fronts, we can use the <font> tag. We can use the font tag to specify the font type, color, and size.

Font Style

The <font> tag is used to display text in different fonts.

<font> tag has three attribute

  1. face
  2. size
  3. color
